My projects

This page outlines my open source development projects.

Current primary projects

  • Wikidata - the free knowledge base that anyone can edit, powering Wikipedia.
  • WMDE Fundraising - fundraising application for Wikimedia Deutschland.
  • Semantic MediaWiki - An extension to MediaWiki hat lets you store and query data within the wiki's pages.

Applications and services

  • QueryR API - a REST compliant webservice for accessing information stored in Wikidata.
  • ReplicatoR - a CLI application to replicate Wikibase entities (mainly from Wikidata).
